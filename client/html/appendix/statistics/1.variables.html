Variables Qualitiative variables can be placed into distinct categories but cannot be ordered or ranked. Quantitative variables can be ordered or ranked. There are two types of quantitative variables:
  1. Discrete variables can assume a finite number of values between two numbers.
  2. Continuous variables can assume an infinite number of values between two numbers.
Qualitative Variable Examples Sex. Eye color. Favorite flavor of ice cream. Discrete Variable Examples Number of outcomes from the roll of a dice. Number of tickets sold at a movie theater. Heart beats per minute. Continuous Variable Examples Height. For example, a person may by 160cm tall, 160.1cm tall, 160.01cm tall, 160.001cm tall, etc. A person's height can take on an infinite number of values between 160cm and 161cm. Other examples are weight, speed, and distance traveled.
