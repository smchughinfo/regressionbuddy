Variance and Standard Deviation Standard Deviation is the average distance from the average. Variance is the square of the average distance from the average. Population Variance
\( \sigma^2 = \frac { \Sigma(X - \mu)^2 } { N } \)
\begin{align} X &= \text{individual value} \\ \mu &= \text{population mean} \\ N &= \text{population size} \end{align}
Sample Variance
\( s^2 = \frac { \Sigma(X - \overline{X})^2 } { n-1 } \)
\begin{align} X &= \text{individual value} \\ \overline{X} &= \text{sample mean} \\ n &= \text{sample size} \end{align}
Sample Variance Shortcut
\( s^2 = \frac { \Sigma X^2 - \frac { \left( \Sigma X \right)^2 } { n } } { n - 1 } \)
\begin{align} X &= \text{individual value} \\ n &= \text{sample size} \end{align}
Population Standard Deviation
\( \sigma = \sqrt{\sigma^2} \)

Sample Standard Deviation

\( s = \sqrt { s^2 } \)
Population Example
3, 6, 1, 3, 2
Find the mean. \( \mu=\tfrac{3+6+1+3+2}{5}=\tfrac{15}{5}=3 \) Subtract the mean from each value.
\(3-3=\phantom{-}0\) \(6-3=\phantom{-}3\) \(1-3=-2\) \(3-3=\phantom{-}0\) \(2-3=-1\)
Square each result.
\(\phantom{-}0^2=0\) \(\phantom{-}3^2=9\) \(-2^2=4\) \(\phantom{-}0^2=0\) \(-1^2=1\)
Sum the squares. \(0+9+4+0+1=14\) Divide by \(N\). \(\ \sigma^2=\tfrac{14}{5}=2.8 \) Take the square root. \( \sigma = \sqrt{2.8} \approx 1.67 \) Sample Example
8, 6, 0, 5, 1
Find the mean. \( \overline{X}=\tfrac{8+6+0+5+1}{5}=\tfrac{20}{5}=4 \) Subtract the mean from each value.
\(8-4=\phantom{-}4\) \(6-4=\phantom{-}2\) \(0-4=-4\) \(5-4=\phantom{-}1\) \(1-4=-3\)
Square each result.
\(\phantom{-}4^2=16\) \(\phantom{-}2^2=4\) \(-4^2=16\) \(\phantom{-}1^2=1\) \(-3^2=9\)
Sum the squares. \(16+4+16+1+9=46\) Divide by \(n-1\). \(\ s^2=\tfrac{46}{4}=11.5 \) Take the square root. \( s = \sqrt{11.5} \approx 3.39 \) Shortcut Example
3, 9, 0, 4, 7
Find the sum of the values. \( \Sigma X = 3+9+0+4+7 = 23 \) Sum the squares of the values. \( \Sigma X^2 = 3^2+9^2+0^2+4^2+7^2 = 155 \) Plug the values into the shortcut formula. \( s^2 = \frac { 155 - \frac { \left( 23 \right)^2 } { 5 } } { 5 - 1 } = \frac { 155 - \frac { 529 } { 5 } } { 4 } = \frac { 155 - 105.8 } { 4 } = \frac { 49.2 } { 4 } = 12.3 \) Take the square root. \( s = \sqrt{12.3} \approx 3.51 \)
