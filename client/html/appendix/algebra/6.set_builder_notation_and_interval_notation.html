Set-Builder Notation and Interval Notation Set-Builder Notation
A few configurations of set-builder notation are listed below:

Enumeration

\( \{ a, b, d \} \)

This set is composed of members \(a\text{,}b\text{, and } d\text{.} \)
Predicate

\( \{x | x \text{ has some condition} \} \)

This set only includes members that meet a condition.
Predicate with Domain

\( \{x \in E| x \text{ has some condition} \} \)

Members of this set are subject to the additional constraint of belonging to another set.
Interval Notation
Interval notation is used to represent all the numbers between two endpoints. An interval between the endpoints \(a\) and \(b\) takes the form \( (a,b) \), \( [a,b) \), \( [a,b] \), or \( (a,b] \). Interval notation has the following properties:
  • Exclusive endpoints use parentheses, ().
  • Inclusive endpoints use brackets, [].
  • Two intervals are joined together with the union operator, \(\cup\).
Enumeration Example

\( \{ 7,-3, 5 \} \)

The set composed of \(7\), \(-3\), and \(5\).
Predicate Example

\( \{x | -3 \leq x \leq 3 \} \)

The set of numbers between \(-3\) and 3, inclusive.
Predicate with Domain Example

\( \{x \in \mathbb{Z}| x > 2 \} \)

The set of integers greater than two.
Interval Notation Example

\( \{x \in E| x \text{ has some condition} \} \)

Members of this set are subject to the additional constraint of belonging to another set.
