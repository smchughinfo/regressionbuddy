The Equation of a Line Set-Builder Notation
A few configurations of set-builder notation are listed below:

Enumeration

\( \{ a, b, d \} \)

This set is composed of members \(a\text{,}b\text{, and } d\text{.} \)
Predicate

\( \{x | x \text{ has some condition} \} \)

This set only includes members that meet a condition.
Predicate with Domain

\( \{x \in E| x \text{ has some condition} \} \)

Members of this set are subject to the additional constraint of belonging to another set.
Interval Notation
Interval notation is used to represent the set of numbers between two endpoints. An interval between the endpoints \(a\) and \(b\) takes the form \( (a,b) \), \( [a,b) \), \( [a,b] \), or \( (a,b] \). Interval notation has the following properties:
  • Exclusive endpoints use parentheses, ().
  • Inclusive endpoints use brackets, [].
  • Two intervals are joined together with the union operator, \(\cup\).
Enumeration Example /images/algebra-appendix/set-builder-example-1.svg https://www.geogebra.org/graphing/wscmhEE2 \( \{ 6,-1, 5 \} \)

The set composed of \(6\), \(-1\), and \(5\).

Predicate Example /images/algebra-appendix/set-builder-example-2.svg https://www.geogebra.org/graphing/Yg2KtG9V \( \{x | -3 \leq x \leq 3 \} \)

The set of \(x\) such that \(x\) is between \(-3\) and \(3\), inclusive.

Predicate with Domain Example /images/algebra-appendix/set-builder-example-3.svg https://www.geogebra.org/graphing/pEEu4ENQ \( \{x \in \mathbb{R}| x > 2 \} \)

The set of \(x\) such that \(x\) is in the set of real numbers that are greater than two.
*Note - \(\in\) stands for in and \(\mathbb{R}\) stands for the set of all real numbers.

Interval Notation Example /images/algebra-appendix/interval-notation-example.svg https://www.geogebra.org/graphing/BxZ4cgWU \( [-4,1)\cup(2,5] \)

The set of numbers between \(-4\) (inclusive) and \(1\) (exclusive) and the set of numbers between \(2\) (exclusive) and \(5\) (inclusive).
