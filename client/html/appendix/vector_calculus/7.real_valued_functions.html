Real-valued Functions Real-valued functions only output real numbers. To express that \(f\) maps \(A\) (a subset of \(\mathbb{R}^n\)) onto \(\mathbb{R}^m\) we write:
\( f:A \subset \mathbb{R}^n \mapsto \mathbb{R}^m \) Scalar-valued Example
\( f: \mathbb{R}^4 \mapsto \mathbb{R} \text{,}\; f(x_1,x_2,x_3,x_4) = x_1+x_2-x_3+x_4 \)
\( f(5, 6, 7, 8) = 5+6-7+8 = 12 \)
Vector-valued Example
\( f: \mathbb{R}^3 \mapsto \mathbb{R}^2 \text{,}\; f(x,y,z) = (x+y,x+z) \)
\( f(1,2,3) = (1+2,1+3) = (3,4) \)
*Note - \(f:\mathbb{R}^4 \mapsto \mathbb{R}\) means "we are using objects in \( \mathbb{R}^4 \) to perform a function that will result in an object in \( \mathbb{R} \)." For example \( f(1,2,3,4) = 1+2-3+4 = 4\).
