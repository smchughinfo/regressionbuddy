Vector Addition and Scalar Multiplication Vector Addition \( \left ( a_1^\phantom{1},\,a_2^\phantom{1},\,a_3^\phantom{1} \right) + \left ( b_1^\phantom{1},\,b_2^\phantom{1},\,b_3^\phantom{1} \right) = \left( a_1^\phantom{1} + b_1^\phantom{1} ,\, a_2^\phantom{1} + b_2^\phantom{1} ,\, a_3^\phantom{1} + b_3^\phantom{1} \right) \) Scalar Multiplication \( k \left ( a_1^\phantom{1},\,a_2^\phantom{1},\,a_3^\phantom{1} \right) = \left ( ka_1^\phantom{1},\,ka_2^\phantom{1},\,ka_3^\phantom{1} \right) \) Vector Addition Example \(\mathbb{R}^2\) /images/vector-calculus-appendix/vector-addition-2d.png https://www.geogebra.org/graphing/atP5PEhv \begin{align*} \boldsymbol{u} &= (-3,3) \\ \boldsymbol{v} &= (4,1) \\ \boldsymbol{u}+\boldsymbol{v} &= (1,4) \end{align*} \(\mathbb{R}^3\) /images/vector-calculus-appendix/vector-addition-3d.png https://www.geogebra.org/3d/QuQrdTNF \begin{align*} \boldsymbol{u} &= (-1,-1,2) \\ \boldsymbol{v} &= (3,0,1) \\ \boldsymbol{u}+\boldsymbol{v} &= (2,-1,3) \end{align*} \(\mathbb{R}^n\)
