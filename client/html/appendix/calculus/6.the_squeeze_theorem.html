The Squeeze Theorem If on some interval \( f(x) \leq g(x) \leq h(x) \) and \( \lim\limits_{x\to a} f(x) = \lim\limits_{x\to a} h(x) = L \) then \( \lim\limits_{x\to a} g(x) = L \). Squeeze Theorem Example /images/calculus-appendix/squeeze-theorem.svg https://www.geogebra.org/graphing/fbZhMDwy

We know that
\( -1 \leq sin\left(\frac{1}{x}\right) \leq 1 \)
and so
\( -x^2 \leq x^2sin\left(\frac{1}{x}\right) \leq x^2 \).
Since
\( \lim\limits_{x\to 0} f(x) = \lim\limits_{x\to 0} h(x) = 0 \)
the
\( \lim\limits_{x\to 0} g(x) = 0 \).
