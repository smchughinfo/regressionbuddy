Inverses of the Trigonometric Functions The inverse trigonometric functions take in radians and output a ratio, effectively undoing the trigonometric functions. Example \( sin \left( \frac { \sqrt{2} } { 2 } \right) = \frac{\pi}{4} \)

\( sin^{-1} \left( \frac{\pi}{4} \right) = \frac { \sqrt{2} } { 2 } \)
