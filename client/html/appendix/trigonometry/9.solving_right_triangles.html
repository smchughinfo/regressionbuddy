Solving Right Triangles All angles and side lengths of a right triangle can be found given one of following conditions:
  1. An angle and a side length are known.
  2. Two side lengths are known.
 Angle and a Side Example /images/trigonometry-appendix/solving-right-triangles-example-1.svg
The first missing side can be found by setting up an equation and using a calculator to find the answer: The first missing angle can be found by setting up an equation and using a calculator to find the answer:
\begin{align} \sin(53.13°) &= \tfrac{\color{#a38f8c}{\text{a}}}{1} \\[2pt] .8 &\approx \tfrac{\color{#a38f8c}{\text{a}}}{1} \\[2pt] .8 &\approx \color{#a38f8c}{\text{a}} \\[5pt] \end{align}
The remaining side, \(\color{#a38f8c}{\text{c}}\), can be found by repeating the same processs or by using the pythagorean theorem.
\begin{align} \color{#a38f8c}{\text{c}} &\approx \sqrt { 1^2 - .8^2 } \approx .6 \end{align}
The missing angle, \( \color{#a38f8c}{\text{C}} \) ,is the complement of the known angle. The remaining side, \(\color{#a38f8c}{\text{e}}\), can be found in the same way as the Angle and a Side Example or by using the pythagorean theorem.
\begin{align} \color{#a38f8c}{\text{C}} &= 90°-53.13°=36.87° \end{align}
Two sides Example /images/trigonometry-appendix/solving-right-triangles-example-2.svg
The first missing angle can be found by setting up an equation and using a calculator to find the answer:
\begin{align} \tan(\color{#a38f8c}{\text{D}}) &= \tfrac{6}{10} \\[2pt] \tan(\color{#a38f8c}{\text{D}}) &= \tfrac{3}{5} \\[2pt] \tan^{-1}\left(\tfrac{3}{5}\right) &= \color{#a38f8c}{\text{D}} \\[2pt] 30.96° &\approx \color{#a38f8c}{\text{D}} \end{align}
The remaining angle, \(\color{#a38f8c}{\text{F}}\), is the complement of \(\color{#a38f8c}{\text{D}}\). The remaining side, \(\color{#a38f8c}{\text{c}}\), can be found by repeating the same processs or by using the pythagorean theorem.
\begin{align} \color{#a38f8c}{\text{F}} &\approx 90°-30.96 \approx 59.04° \end{align}
The remaining side, \(\color{#a38f8c}{\text{e}}\), can be found in the same way as the Angle and a Side Example or by using the pythagorean theorem.
\begin{align} \color{#a38f8c}{\text{e}} &= \sqrt { 6^2+10^2 } = 2\sqrt{34} \end{align}
