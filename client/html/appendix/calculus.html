Limit Theorems

  1. \( \lim\limits_{x\to a} c\cdot f(x) = c\cdot \lim\limits_{x\to a} f(x) \)
    *\(c\) is a constant.
  2. \( \lim\limits_{x\to a} \left[f(x)\right]^{r} = \left[\lim\limits_{x\to a} f(x)\right]^{r} \)
    *\(r\) is a positive constant.
  3. \( \lim\limits_{x\to a} \left[f(x)+g(x)\right] = \lim\limits_{x\to a} f(x) + \lim\limits_{x\to a} g(x) \)
  4. \( \lim\limits_{x\to a} \left[f(x)-g(x)\right] = \lim\limits_{x\to a} f(x) - \lim\limits_{x\to a} g(x) \)
  5. \( \lim\limits_{x\to a} \left[f(x) \cdot g(x)\right] = \lim\limits_{x\to a} f(x) \cdot \lim\limits_{x\to a} g(x) \)
  6. \( \lim\limits_{x\to a} \frac{f(x)}{g(x)} = \frac{\lim\limits_{x\to a} f(x)}{\lim\limits_{x\to a} g(x)} \).
    *\(g(x)\neq 0\)

Theorem 1 Example \( \lim\limits_{x\to\ 2} \frac{1}{3}x = \frac{1}{3}(2) = \frac{2}{3} \)
Theorem 2 Example \( \lim\limits_{x\to\ 5} x^2 = 5^2 = 25\)
Theorem 3 Example \( \lim\limits_{x\to\ 3} x^2 + x = 3^2 + 3 = 9 + 3 = 12 \)
Theorem 4 Example \( \lim\limits_{x\to\ 2} x^2 - x = 4 - 2 = 2 \)
Theorem 5 Example \( \lim\limits_{x\to\ \frac{\pi}{2}} xsin(x) = \frac{\pi}{2}sin(\frac{\pi}{2}) = \frac{\pi}{2}(1) = \frac{\pi}{2} \)
Theorem 6 Example \( \lim\limits_{x\to\ 0} \frac{cos(x)}{\sqrt{1-x}} = \frac{cos(0)}{\sqrt{1-0}} = \frac{1}{1} = 1 \)

Evaluating Complicated Limits

Sometimes you have to do a little algebra before direct substitution will work.

Factoring Example \( \lim\limits_{x\to\ 3} \frac{x^2-9}{x-3} = \lim\limits_{x\to\ 3} \frac{(x+3)(x-3)}{x-3} = \lim\limits_{x\to\ 3} x+3 = 3 + 3 = 6 \)
Conjugate Example \( \lim\limits_{x\to\ 25} \frac{5-\sqrt{x}}{25-x} = \lim\limits_{x\to\ 25} \frac{5-\sqrt{x}}{25-x}\frac{5+\sqrt{x}}{5+\sqrt{x}} = \lim\limits_{x\to\ 25} \frac{25-x}{(25-x)(5+\sqrt{x})} = \lim\limits_{x\to\ 25} \frac{1}{5+\sqrt{x}} = \frac{1}{5+\sqrt{25}} = \frac{1}{5+5} = \frac{1}{10} \)
Divide by Highest Power Example \( \lim\limits_{x\to\ \infty} \sqrt{\frac{5x^2}{x^2+2}} = \lim\limits_{x\to\ \infty} \frac{\sqrt{5x^2}}{\sqrt{x^2+2}} = \lim\limits_{x\to\ \infty} \frac {\frac {\sqrt{5x^2}} {x} } {\frac {\sqrt{x^2+2}} {x} } = \lim\limits_{x\to\ \infty} \frac { \sqrt{\frac{5x^2}{x^2}} } { \sqrt{\frac{x^2+2}{x^2}} } = \lim\limits_{x\to\ \infty} \frac { \sqrt{\frac{5x^2}{x^2}} } { \sqrt{\frac{x^2}{x^2} + \frac{2}{x^2}} } = \lim\limits_{x\to\ \infty} \frac { \sqrt{5} } { \sqrt{1 + \frac{2}{x^2}} } = \frac { \sqrt{5} } { \sqrt{1 + \frac{2}{\infty}} } = \frac { \sqrt{5} } { \sqrt{1 + 0} } = \frac { \sqrt{5} } { \sqrt{1} } = \sqrt{5} \)
