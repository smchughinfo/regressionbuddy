\(\mathbb{R}^n\)
\(\mathbb{R}^1\), \(\mathbb{R}^2\), \(\mathbb{R}^3\), \(\mathbb{R}^4\), etc. are generally written as \(\mathbb{R}^n\). That is, \(\mathbb{R}^n\) may refer to \(\mathbb{R}^1\), \(\mathbb{R}^2\), \(\mathbb{R}^3\), \(\mathbb{R}^4\), etc.

Vector Addition and Scalar Multiplication

Vector Addition \( \left ( a_1^\phantom{1},\,a_1^\phantom{1},\,a_3^\phantom{1} \right) + \left ( b_1^\phantom{1},\,b_2^\phantom{1},\,b_3^\phantom{1} \right) = \left( a_1^\phantom{1} + b_1^\phantom{1} ,\, a_2^\phantom{1} + b_2^\phantom{1} ,\, a_3^\phantom{1} + b_3^\phantom{1} \right) \)
Scalar Multiplication \( k \left ( a_1^\phantom{1},\,a_1^\phantom{1},\,a_3^\phantom{1} \right) = \left ( ka_1^\phantom{1},\,ka_1^\phantom{1},\,ka_3^\phantom{1} \right) \)

Example \(1+1=2\)

Properties of Vector Addition and Scalar Multiplication

The following properties of vector spaces can be derived from vector addition and scalar multiplication.:

Associative \( \left(kl\right) \left( a_1,\,a_2,\,a_3 \right) = k \left[ l \left( a_1,\,a_2,\,a_3 \right) \right] \)
Distributive \( \left(k+l\right) \left( a_1,\,a_2,\,a_3 \right) = k \left( a_1,\,a_2,\,a_3 \right) + l \left( a_1,\,a_2,\,a_3 \right) \)
\( k \left[ \left( a_1,\,a_2,\,a_3 \right) + \left( b_1,\,b_2,\,b_3 \right) \right] = k \left( a_1,\,a_2,\,a_3 \right) + k \left( b_1,\,b_2,\,b_3 \right) \)
Identity Properties \( k \left( 0,0,0 \right) = \left( 0,0,0 \right) \)
\( 0 \left( a_1,\,a_2,\,a_3 \right) = \left( 0,0,0 \right) \)
\( 1 \left( a_1,\,a_2,\,a_3 \right) = \left( a_1,\,a_2,\,a_3 \right) \)
