Elementary Row Operations

There are three elementary row operations which can be used to produce a different, but equivalent, system of equations. They are:

  1. Interchange two equations.
  2. Multiply an equation by a nonzero constant.
  3. Add one equation to another.
Examples Interchange Two Equations
0 1 7 -5; -3 3 0 2; 2 -1 0 8 \( \phantom{ \begin{matrix} \color{#a38f8c}{R_1 \leftrightarrow R_2} \\ \\ \\ \end{matrix} } \)
-3 3 0 2; 0 1 7 -5; 2 -1 0 8 \( \begin{matrix} \color{#a38f8c}{R_1 \leftrightarrow R_2} \\ \\ \\ \end{matrix} \)
Multiply by a Nonzero Constant
7 0 -1 4; 0 0 5 1; 9 -3 0 6 \( \phantom { \begin{matrix} \\ \\ \color{#a38f8c}{\frac{1}{3}R_3 \rightarrow R_3} \end{matrix} } \)
7 0 -1 4; 0 0 5 1; 3 -1 0 2 \( \begin{matrix} \\ \\ \color{#a38f8c}{\frac{1}{3}R_3 \rightarrow R_3} \end{matrix} \)
Add One Equation to Another
7 -4 2 3; 6 -3 0 -1; -6 4 -2 1 \( \phantom{ \begin{matrix} \color{#a38f8c}{R_3+R_1 \rightarrow R_1} \\ \\ \\ \end{matrix} } \)
1 0 0 4; 6 -3 0 -1; -6 4 -2 1 \( \begin{matrix} \color{#a38f8c}{R_3+R_1 \rightarrow R_1} \\ \\ \\ \end{matrix} \)
