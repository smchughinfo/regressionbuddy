The Number of Solutions to a System of Linear Equations A solution of a system linear equations is a point that every equation in the system touches.

A system of linear equations can have no solutions, one solution, or an infinite number of solutions.

  1. Systems with no solutions are called inconsistent.
  2. Systems that have a solution are called consistent.

If \(x_4^\phantom{1}=1\) and \(x_3^\phantom{1}=2\) there is no way \(x_3^\phantom{1}+x_4^\phantom{1}\) can equal \(4\), so the system is inconsistent.

One Solution \begin{align*} x_1^\phantom{1} + x_2^\phantom{1} + x_3^\phantom{1} + x_4^\phantom{1} + x_5^\phantom{1} & = 1 \\ \phantom{x_1^1+}x_2^\phantom{1} + x_3^\phantom{1} + x_4^\phantom{1} + x_5^\phantom{1} &= 2 \\ \phantom{x_1^1+x_2^1}x_3^\phantom{1} + x_4^\phantom{1} + x_5^\phantom{1} &= 8 \\ \phantom{x_1^1-x_2^1+x_3^1}x_4^\phantom{1} + x_5^\phantom{1} &= 4 \\ \phantom{x_1^1-x_2^1+x_3^1+x_4^1}x_5^\phantom{1} &= 2 \\ \end{align*}

If \(x_5^\phantom{1}=2\) then \(x_4^\phantom{1}=2\), \(x_3^\phantom{1}=4\), \(x_2^\phantom{1}=-6\), and \(x_1^\phantom{1}=-1\) so the system has only one solution.

Infinite Solutions \begin{align*} x_1^\phantom{1} + x_2^\phantom{1} + x_3^\phantom{1} + x_4^\phantom{1} + x_5^\phantom{1} & = 1 \\ \phantom{x_1^1+}x_2^\phantom{1} + x_3^\phantom{1} + x_4^\phantom{1} + x_5^\phantom{1} &= 2 \\ \phantom{x_1^1+x_2^1}x_3^\phantom{1} + x_4^\phantom{1} + x_5^\phantom{1} &= 8 \\ \phantom{x_1^1-x_2^1+x_3^1}x_4^\phantom{1} + x_5^\phantom{1} &= 4 \\ \phantom{x_1^1-x_2^1+x_3^1+x_4^1}x_5^\phantom{1} &= t \end{align*}

In this case \(x_5^\phantom{1}\) equals the variable \(t\) . Since the values of the other variables change as \(x_5^\phantom{1}\) changes this system has an infinite number of solutions.
