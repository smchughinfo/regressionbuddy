Linear Algebra Appendix


Linear Equations in n Variables

Two Variables \(Ax+By=C\)
The line crosses the x axis at \(\frac{C}{A}\) when \(A \neq 0\). The line crosses the y axis at \(\frac{C}{B}\) when \(B \neq 0\). The slope of the line is \(-\frac{A}{B}\).
Three Variables \(Ax+By+Cz=D\)
The vector \((A,B,C)\) is normal (orthogonal, at a 90 degree angle) to the plane. The distance from the origin to point on the plane closest to the origin is \( \frac{\left|D\right|}{\sqrt{A^2+B^2+C^2}} \).

Two Variables Example
\(4x+2y=4\)
Three Variables Example
\(2x+2y+2z=4\)

Linear Equations in n Variables

A solution of a system linear equations is a point that every equation in the system touches.

A system of linear equations can have no solutions, one solutions, or an infinite number of solutions.

  1. Systems with no solutions are called inconsistent.
  2. Systems that have a solution are called consistent.

Two Variables Example
No Solutions
\(\hphantom{\-}x-y=1\)
\(-x+y=1\)
One Solution
\(\hphantom{2}x-\hphantom{2}y=1\)
\(2x-2y=2\)
Infinite Solutions
\(\hphantom{-}x+\hphantom{3}y=2\)
\(-x+3y=2\)
Three Variables Example
No Solutions
\(x+y+z=1\)
\(x+y+z=\frac{5}{2}\)
\(x+y+z=5\)
One Solution
\(\hphantom{-}x+y+z=1\)
\(\hphantom{-}x+y-z=1\)
\(-x+y\hphantom{-zz}=5\)
Infinite Solutions
\(\hphantom{-}x+\hphantom{3}y=2\)
\(-x+3y=2\)
