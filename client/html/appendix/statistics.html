Addition Property of Equality

"Adding a number to both sides of the equation doesn't change the solution."

If A, B and C are real numbers then

\(\textbf{A = B}\) and \(\textbf{A + C = B + C}\)

are equivelant.

\(X - 5 = 11\)
\(X - 5\textbf{ + 5} = 11\textbf{ + 5}\)
\(X = 16\)
