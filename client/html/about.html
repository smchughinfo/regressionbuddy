About

This site is for anyone that knows the math and wants some practice problems to stay sharp. This site is not intended to teach math, it's just for practice. Reddit loved the idea so I made it. New problems are published every Sunday morning. Enjoy!

LaTeX

Disqus, the comment system used on this site, does not support LaTeX. They do support images so you could use an online latex editor, take a screenshot, crop it, and upload that.

Contribute

Next week's post can be reviewed here. If you see a bug you can shoot me an email. You can submit a pull request here.
